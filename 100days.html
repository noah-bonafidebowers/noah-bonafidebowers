11/29/24

CONCLUSION:

So! Turns out doing anything (especially something like this with an un-restricted time limit) for 100 days can be… a lot! I think getting a quarter of the way there, 25 days, at least was an achievement within itself, and taught me a lot. The biggest and best thing I learned from this would be that I really don’t have that many ideas. The reason I started this project was because I felt that whenever I would sit down to make a project, it would go south because I would either keep adding new ideas, deviating from the original focus, or create a new idea entirely; however, doing this project showed me that these ideas actually all have much in common, so I’m more comfortable with going forward with one project. On that note, I have become much more focused on my manga creation. You can see my progress for my ideas on my Instagram @theNoahBowers (find on the homepage!). I realized a lot of the work here would be best done in that style, and I’ve always been inspires by comics and animation as a kid and manga and anime in more recent years, so I can really see myself becoming a part of this industry! Now I am doing one shots in a similar way to this 100 days concept, but weekly and not a set end limit since I can see myself doing this stuff forever. Thanks so much for reading if you made it this far, I love you.
